# Netflix Clone MERN

[![Demo](https://netlify.app)](https://netflix-askhan.netlify.app/)
  
## Overview

This project is a Netflix clone developed using the MERN stack (MongoDB, Express.js, React, Node.js). It includes features such as user authentication and a main page showcasing available content.

## How to Run

1. **Fork the Repository**

2. **Front-end Setup:**
   - Navigate to the `netflix-ui` directory
   - Run `npm install` to install dependencies
   - Run `npm start` to launch the front-end

3. **Back-end Setup:**
   - Navigate to the `backend` directory
   - Run `npm install` to install dependencies
   - Run `node index.js` to start the back-end server
